# My Node.js Project

A simple Node.js project with Express.js server.

## Установка

```bash
npm install
```

## Запуск

### Режим разработки (с автоматическим перезапуском)
```bash
npm run dev
```

### Производственный режим
```bash
npm start
```

## Использование

После запуска сервера, откройте браузер и перейдите по адресу:
- `http://localhost:3000` - главная страница
- `http://localhost:3000/api/health` - проверка состояния сервера

## API Эндпоинты

- `GET /` - Возвращает приветственное сообщение
- `GET /api/health` - Возвращает информацию о состоянии сервера

## Структура проекта

```
my-nodejs-project/
├── index.js          # Главный файл сервера
├── package.json      # Конфигурация проекта
└── README.md        # Этот файл
```

## Настройка порта

По умолчанию сервер запускается на порту 3000. Вы можете изменить порт, установив переменную окружения PORT:

```bash
PORT=8080 npm start
```
